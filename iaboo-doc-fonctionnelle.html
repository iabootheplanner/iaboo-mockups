01 — Introduction

⊡
Vue d'ensemble

IABOO est un portail de gestion de la planification des agents de sécurité privée, piloté par une intelligence artificielle qui automatise les tâches répétitives.

🤖 Ce que fait l'IA
  • Envoyer la collecte des disponibilités mensuelles (WhatsApp)
  • Générer le planning mensuel (contraintes légales + contrat)
  • Chercher un remplaçant dès qu'une absence est signalée
  • Traiter les demandes clients envoyées par email
  • Créer des anomalies et alerter si aucune solution n'est trouvée
👤 Ce que vous faites
  • Superviser et valider les décisions de l'IA
  • Intervenir manuellement en cas d'anomalie non résolue
  • Gérer le référentiel (agents, sites, clients)
  • Configurer les paramètres du système
  • Relancer les agents n'ayant pas répondu

04 — Page

👤
Agents

Planificateur Admin

Recherche & filtres

Barre de recherche : tapez un nom, prénom ou numéro de téléphone. La liste se filtre instantanément (pas besoin d'appuyer sur Entrée).

Filtres rapides : cliquez sur les chips pour filtrer par type (Rattaché / Volant), par statut de carte pro (Valide / Expire bientôt / Expirée) ou par site.

Fiche détail d'un agent

Cliquez sur une ligne du tableau pour ouvrir le panneau de détail à droite. Il contient :

  • Coordonnées (téléphone, email, numéro WhatsApp)
  • Type de contrat et heures mensuelles (effectuées / total contrat)
  • Numéro et date d'expiration de la carte professionnelle
  • Compétences : SSIAP1, SSIAP2, SSIAP3...
  • Jours fixes, préférences, conflits avec d'autres agents

En bas du panneau : Voir planning → planning de cet agent · Modifier → formulaire d'édition.

Ajouter, modifier, désactiver

1
Ajouter un agent

Cliquez sur + Ajouter un agent en haut à droite. Remplissez le formulaire (identité, type, site, heures contrat, carte pro, compétences). L'agent sera inclus dans la prochaine collecte de disponibilités.

2
Modifier un agent

Depuis sa fiche détail, cliquez sur Modifier. Le formulaire est pré-rempli. Un message d'avertissement apparaît si la modification impacte des missions planifiées.

3
Renouveler une carte pro

Depuis la fiche d'un agent avec une alerte carte, cliquez sur Mettre à jour la carte. Saisissez le nouveau numéro et la nouvelle date d'expiration. L'alerte disparaît immédiatement.

4
Désactiver un agent

Cliquez sur l'icône × dans la liste. Une confirmation est demandée. L'agent n'apparaît plus dans les plannings futurs mais son historique est conservé.

Replanifier avec l'IA

Le bouton 🔄 Replanifier (vert, en haut à droite) déclenche une nouvelle génération du planning mensuel. Utilisez-le après avoir modifié des disponibilités ou des contrats de site.

🤖
Ce que l'IA respecteDisponibilités déclarées · Heures de contrat · Repos 11h minimum · 6 jours max consécutifs · 35h/semaine · Jours fixes des agents · Compétences requises par site · Conflits interpersonnels.

Résolution manuelle

1
Ouvrir l'anomalie

Cliquez sur l'anomalie dans la liste. Consultez la chronologie et les agents déjà contactés.

2
Consulter les propositions IA

Si l'IA a identifié des solutions, elles apparaissent avec un bouton "Appliquer" pour chacune.

3
Résoudre manuellement

Cliquez sur Résoudre manuellement, sélectionnez un agent disponible, confirmez. Le planning est mis à jour et l'agent reçoit une notification WhatsApp.

08 — Page

🗓️
Disponibilités

Planificateur Admin

Collecte automatique mensuelle

Chaque mois, à la date configurée dans les Paramètres (défaut : le 20), l'IA envoie automatiquement un message WhatsApp à tous les agents pour recueillir leurs indisponibilités.

💬
Exemple de réponse agent"Bonjour, pour juillet je serai pas dispo les 5, 6 et 12. Le reste ça va. Merci" — L'IA parse automatiquement ce message et enregistre les jours 5, 6 et 12 comme indisponibles.

Le bandeau de progression en haut de l'écran indique le nombre d'agents ayant répondu, la date de la dernière relance, et une barre de progression.

La grille centrale (agents × jours) donne une vue synthétique : vert = disponible · rouge "OFF" = indisponible · pointillé = pas encore répondu.

Saisie & correction manuelle

  • Corriger — cliquez sur ✏ Modifier dans le panneau d'un agent. Le calendrier devient interactif : cliquez sur un jour pour basculer disponible ↔ indisponible.
  • Saisir — cliquez sur ✏ Saisir manuellement dans le header pour ouvrir le formulaire (sélection agent + calendrier interactif).
  • Relancer individuellement — depuis le panneau d'un agent en attente, bouton 📨 Relancer cet agent.
  • Relancer en masse — bouton 📨 Relancer tous en haut de page. Une confirmation liste les agents concernés.
ℹ️
Un délai minimum de 24h est respecté entre deux relances au même agent pour éviter le spam.

Lancer la planification

Une fois le seuil minimum de réponses atteint (défaut : 80%, configurable dans Paramètres), le bouton 🚀 Lancer planification s'active.

Une fenêtre de confirmation résume la situation. Vous êtes ensuite redirigé vers l'écran Plannings pour vérifier le résultat.

11 — Référence

💬
Questions fréquentes

Un agent n'a pas répondu à la collecte. Que faire ? ›
Allez dans l'écran Disponibilités, localisez l'agent (badge ⏳ En attente), et utilisez 📨 Relancer cet agent pour lui envoyer un rappel WhatsApp. Si toujours pas de réponse après 24h, saisissez ses disponibilités manuellement après l'avoir contacté par téléphone.
L'IA n'a pas trouvé de remplaçant. Que faire ? ›
Une anomalie "Remplacement impossible" est créée automatiquement et apparaît dans l'écran Anomalies. Ouvrez-la, consultez les propositions de l'IA, et utilisez Résoudre manuellement pour affecter un agent de votre choix.
Le planning généré ne me convient pas. Peut-on le corriger ? ›
Oui. Modifiez les disponibilités d'un agent manuellement dans l'écran Disponibilités, puis cliquez sur 🔄 Replanifier dans l'écran Plannings pour régénérer le planning avec les nouvelles données.
Comment ajouter un nouveau site pour un client existant ? ›
Dans l'écran Sites & Clients, sélectionnez le client dans la colonne gauche, puis cliquez sur + Nouveau site. Renseignez toutes les informations du site (adresse, postes requis, horaires, compétences, dates de contrat) et sauvegardez.
Un client envoie des demandes par email. Comment est-ce traité ? ›
L'IA surveille l'adresse configurée dans Paramètres → Intégrations (champ "Email de collecte clients"). Dès réception, l'IA parse la demande (dates, nombre d'agents, compétences) et lance automatiquement la recherche de disponibilités — même processus qu'un remplacement d'urgence.
Comment vérifier qu'un message WhatsApp a bien été traité ? ›
Consultez le Journal d'activité dans Paramètres → Journal. Chaque message reçu et parsé apparaît avec son horodatage, le résultat du traitement et l'auteur (IA Auto ou initiales du planificateur).
Comment vérifier que tous les agents ont reçu leur planning ? ›
Dans l'écran Plannings, les plannings avec statut "Envoyé" ont bien été transmis par email. Vous pouvez aussi renvoyer le mail manuellement à un agent spécifique depuis la vue par agent via le bouton Renvoyer le mail.
